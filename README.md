#### 工程结构

- APP：壳工程
- module_chat：聊天模块
- module_common：通用业务模块（继承自module_lib）
- module_learn：学习模块

- module_live：直播模块
- module_login：登录模块
- module_main：入口模块模块
- module_music：音乐模块
- module_news：新闻模块
- module_pay：支付模块
- module_share：分享模块
- module_video：视频模块
